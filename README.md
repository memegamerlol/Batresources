# Batresources

A comprehensive collection of resources for Windows batch file programming. This repository contains examples, utilities, and documentation to help you learn and master batch scripting.

### Examples
Organized by difficulty level to help you learn progressively:

- **[Basic Examples](examples/basic/)** - Fundamental concepts
  - `hello_world.bat` - Your first batch file
  - `variables.bat` - Working with variables
  - `loops.bat` - Loop structures
  - `conditionals.bat` - If statements and conditions

- **[Intermediate Examples](examples/intermediate/)** - Building on the basics
  - `file_operations.bat` - Reading, writing, and managing files
  - `user_input.bat` - Getting and validating user input
  - `string_manipulation.bat` - String operations and formatting

- **[Advanced Examples](examples/advanced/)** - Complex techniques
  - `functions.bat` - Creating reusable functions
  - `error_handling.bat` - Proper error management
  - `menu_system.bat` - Interactive menu interface

### Utilities
Practical, ready-to-use batch scripts:

- `system_info.bat` - Display comprehensive system information
- `backup.bat` - Simple backup utility with timestamps
- `network_tools.bat` - Network diagnostics and troubleshooting
- `cleanup.bat` - Clean temporary files and folders

## üöÄ Quick Start

1. **Download** any example file from the repository
2. **Open** it in a text editor to see the code
3. **Double-click** the file to run it (or run it from Command Prompt)
4. **Modify** the code and experiment!

## üí° Usage Tips

- Always start scripts with `@echo off` for cleaner output
- Use `pause` at the end to see results before window closes
- Test scripts in a safe directory first
- Read the comments in each example to understand the code
